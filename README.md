# Personal Portfolio Website (GSAP Animated)

A modern multi-page personal portfolio website built using **HTML, CSS, and JavaScript** with smooth **GSAP animations**.  
This project highlights skills, featured projects, and provides a contact page for easy communication.

---

## Overview

This portfolio website is designed to present a clean professional profile with:
- animated UI elements
- responsive layout
- smooth scroll-based effects

It includes separate pages for **Home**, **Skills**, **Projects**, and **Contact**.

---

## Key Highlights

- Multi-page portfolio website
- Smooth animations using GSAP + ScrollTrigger
- Typing animation on homepage
- Skills progress bars with animation
- Projects section with tags & action buttons
- Responsive design for mobile and desktop

---

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)

---

## Project Pages

| Page | Description |
|------|-------------|
| Home | Intro section + typing animation |
| Skills | Skill cards + progress levels |
| Projects | Featured projects with UI cards |
| Contact | Contact form layout |

## How To Run Locally

1. Download / Clone this repository
2. Open `index.html` in any browser
   *(Recommended: Use VS Code Live Server for best experience)*

---

## Customization

To update this portfolio for your profile:

* Change name and content inside `index.html`
* Update skills in `skills.html`
* Add/replace projects in `projects.html`
* Update contact form details in `contact.html`
* Modify UI colors and styling in `style.css`
